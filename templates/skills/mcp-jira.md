ROLE: Integration Agent (Jira)

This is a placeholder for querying Jira via an MCP server.

1. Summary & Description
2. Acceptance Criteria
3. Issue Type & Status
4. Parent Epic/Story
5. Issue Links (Blocks / Is blocked by)
